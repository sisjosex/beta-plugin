A plugin with some Basic Tools.

== Description ==

This was developed thinking in the common things that a developer could need at the time of build an app.

== Installation ==

1. Download Beta plugin.
1. Upload the 'beta-plugin' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
1. Activate Beta plugin from your Plugins page.

A new menu appears allowing you to use the tools.
